# DAPP Starter Kit

> 基于星云链的DAPP脚手架，帮助你快速开发区块链应用。
## 功能
1. 发布数据到星云公链；
2. 从星云公链读取数据；
3. 用户/管理员可删除自己发布的数据；
4. 管理员可新增与编辑分类。

## 开始

 - Clone or download this repository
 - Enter your local directory, and install dependencies:

``` bash
npm install
```

## Develop

``` bash
# serve with hot reload at localhost:8011
npm run dev
```

## Build

``` bash
# build for production with minification
npm run build
```
